# VIBE 맥

생성

```shell
./create.sh
```

# 특징

* 미디어 키 지원
* 기타 기능 지원

## 글로벌 단축키

1. 재생/일시정지 토글 : 재생/일시정지 키
2. 다음 트랙 : 다음 트랙 키
3. 이전 트랙 : 이전 트랙 키
4. 현재 재생 중인 곡 보기 : Ctrl + Shift + 2
5. 좋아요 체크 및 해제 : Ctrl + Shift + 1

## 앱 내 단축키

1. 재생/일시정지 토글 : P
2. 재생 : U
3. 일시정지 : I
4. 좋아요 체크 및 해제 : J
5. 다음 트랙 : ]
6. 이전 트랙 : [
